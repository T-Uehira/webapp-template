# webapp-template

## Getting Started

	$ npm install
	$ npm start
	$ npm test

## Principal Platforms & Libraries

* Frontend:
	* AngularJS v1
	* Skyblue CSS Framework v1
* Backend:
	* Node.js v5
	* Express v4
* Package Management:
	* npm (for backend)
	* Bower v1 (for frontend)
* Test:
	* ESLint v2
